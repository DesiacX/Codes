##Manual Clone Soundbank Unload Fix + Automatic EX Soundbank Unload Fix 

This code is based on the LXP Modification of the this code (https://smashboards.com/threads/ds22s-code-showcase.428060/#post-20793302).

This version has 2 functions. If you know which characters share the same soundbank, you can have the code manually handle them. If you do not, for EX Characters the process will occur automatically.
